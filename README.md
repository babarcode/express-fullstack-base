# Express Fullstack Base

A boilerplate application for building web apps using express.
 


## Tech Stack

**Client:** EJS, Bootstrap

**Server:** Node, Express, Mongoose JS


## Installation

```bash
  npm install
```
    
## Seeding Admin User

```bash
  node db/seeding.js

  user: admin
  pass: admin
```

## Run App

```bash
  npm start
```
